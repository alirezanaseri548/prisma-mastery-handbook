# Prisma Mastery Handbook

جزوه آموزشی جامع **Prisma** به زبان فارسی برای توسعه Backend با **Node.js** و **TypeScript**.

## معرفی
این مخزن شامل یک جزوه آموزشی کامل درباره Prisma است که به صورت مرحله‌به‌مرحله مفاهیم اصلی و کاربردی را آموزش می‌دهد؛ از راه‌اندازی اولیه تا طراحی Schema، مدیریت روابط، Migration، عملیات CRUD و الگوهای پیشرفته Backend.

## سرفصل‌ها
- معرفی Prisma
- راه‌اندازی پروژه
- طراحی Schema
- مدیریت روابط بین مدل‌ها
- Migration
- Prisma Client
- عملیات CRUD
- کوئری‌های پیشرفته
- الگوهای Backend
- نکات و چک‌لیست نهایی
